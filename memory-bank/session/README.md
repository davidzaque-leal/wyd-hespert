# Session Memory Bank

This directory contains temporary notes and information related to the current development session for the Fleet Ticket Validator project.

## Purpose

The session directory is used for:

1. Temporary notes during development
2. Current session goals and progress tracking
3. Debugging information
4. Ideas and thoughts that haven't been formalized yet

Unlike the persistent directory, files here are meant to be ephemeral and may be cleared between sessions.

## Usage

- Create markdown files for specific tasks or topics
- Use consistent naming conventions: `YYYY-MM-DD-topic.md`
- Delete files when they're no longer needed
- Move important information to the persistent directory when appropriate
